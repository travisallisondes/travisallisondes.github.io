※本製品単体では使用できません。ライブラリを使用するには、VOCALOID4 EditorまたはVOCALOID Editor for Cubeseが必要です。

アーティスト「GACKT」のパワフルな声の部分をベースに制作したVOCALOID4専用歌声ライブラリです。
「アタックが強くはっきりとした迫力がある」GACKTの声質でボーカルの歌声を作成できます。

◆データベースの再調整により、自然な言葉のつながりやノイズ軽減など、さらに高音質な歌声合成を実現します。

◆VOCALOID4の特長でもある声を激しくふるわせるような効果「グロウル」や他のライブラリ「NATIVE」、「WHISPER」と同時の使用でライブラリをブレンドしオリジナルライブラリが作れる「クロスシンセシス」機能も使用可能です。


■ 動作環境 ■
【Windows】

  • OS：Windows 11、Windows 10（64bit/32bit）
  • CPU：Intel Dual Core CPU
  • メモリ：2GB以上
  • HDD：
    ・2GB以上（VOCALOID4 Editorと使用の場合）
    ・10GB以上（VOCALOID4 Editor for CubaseとCubase使用の場合）
  • その他：DVD-ROMドライブ、オーディオデバイス
【Mac】
  • OS：Mac OS 12、Mac OS 11、Mac OS X（10.8～10.15）
  • CPU：Intel Dual Core CPU
  • メモリ：2GB以上
  • HDD：
    ・2GB以上（VOCALOID4 Editorと使用の場合）
    ・10GB以上（VOCALOID4 Editor for CubaseとCubase使用の場合）
  • その他：DVD-ROMドライブ、オーディオデバイス
※アクティベーションならびに最新バージョンのアップデートを行うためにコンピュータがインターネット環境に接続されている必要があります。
※上記の動作条件を満たしている場合でも、全てのコンピュータにおける動作を保証するものではありません。
※コンピュータの総合的な性能により同時に使用可能なトラック数などパフォーマンスに違いがあります。
