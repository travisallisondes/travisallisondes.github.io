Windows、Linux、Android、MacOS用 IMX179 8MP オートフォーカスUSBカメラ、シングルマイク付き

概要

  • IMX179 8MP オートフォーカスUSBカメラモジュール (シングル マイク付き)
  • 30fps時、最大解像度 3264 (H) x 2448 (V) の高品質画像を提供
  • 利便性と効率性を高めるオートフォーカスレンズを搭載
  • Windows、Linux、Android、Macなどのさまざまなオペレーティングシステムに対応
  • 使い勝手の良い設計で、操作に追加のドライバは不要

Windows、Linux、Android、Mac OS用 IMX179 8MP オートフォーカスUSBカメラモジュール (シングル マイク付き) は、8MP IMX179イメージセンサを中心に構築された高品質のカメラモジュールです。30フレーム/秒時、最大解像度3264 (H) x 2448 (V)で、MJPG/YUY2出力フォーマットの画像を出力することができます。

このカメラモジュールの主な特徴のひとつは、カメラ機能の利便性と効率を大幅に向上するオートフォーカスレンズです。イメージセンサの適用範囲が拡大し、幅広いプロジェクトに活用できるようになりました。

カメラモジュールは、63°（H）のオートフォーカスレンズを搭載し、特定の用途に適しています。Windows、Linux、Android、Mac OSを含む様々なOSに対応するように設計してあります。このカメラモジュールは使い勝手が良くで、操作に追加ドライバは不要です。

仕様

  • センサ：IMX179
  • 解像度：8MP
  • フィルタ：カラー
  • シャッタ：ローリングシャッタ
  • 光学サイズ：1/3.2インチ
  • 出力形式：YUYV/MJPG
  • センサアクティブエリア：3264 (H) x 2448 (V)
  • フレームレート
    • MJPG：
      • 30fps@ 3264 x 2448 / 2592 x 1944 / 1920 x 1080 / 1600 x 1200 / 1280 x 720 / 640 x 480
    • YUY2：
      • 1fps @ 3264 x 2448
      • 2fps@ 2592 x 1944
      • 5fps @ 1920 x 1080 / 1600 x 1200
      • 10fps@1280 x 720
      • 30fps @ 640 x 480
  • フォーカス：オートフォーカス
  • FOV：63° (高さ)
  • レンズマウント: M8.5 ± 0.25
  • F.NO：2.0±5%
  • EFL：3.543
  • インタフェース：USB 2.0デバイス
  • コネクタ：B4B-ZR
  • UVC準拠：追加ドライバ不要
  • 動作電圧：DC 5V +/-5%
  • 動作電流：最大300mA
  • 動作温度：-20～70 °C (-4～158 °F)
  • オーディオ：シングルマイク
  • 対応OS：Windows、Linux、Android、Mac OS
  • サンプルアプリケーション：AMCap-ビデオ および 静止画キャプチャ用Windowsカメラアプリケーション
  • UVCコントロール：明度、コントラスト、色相、彩度、シャープネス、ガンマ、ホワイトバランス、バックライト補正、ゲイン
