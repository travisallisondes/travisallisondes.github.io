ADL アルファデザインラブス GT-C2 NCF　0.14m　Type C to Cケーブル GTC2NCF

ハイエンドグレード Type C to C USB 2.0 cable
ケーブル長： 14cm（ストレートタイプ）

昨今USB Type-C端子は、様々なデバイスで使用されております。

ポータブルヘッドホンアンプ、ドングル、スマートフォンやタブレットなどに合うよう、洗練されたフォルムはもちろん、どの方向にも対応できるようケーブルは取り回しやすい作りになっております。Type-Cコネクタの魅力は、データ通信と給電を同時に行うことを可能とする事です。今後様々な製品や規格に採用されていきますので、更に普及していく事が予想されているコネクターです。


■NCFシェル（マルチマテリアルハイブリッド構造）
静電気対策として特殊素材「NCF」を調合。強力な制振効果に加え、静電効果が高まりました。

■銅銀合金線素材を採用
トランスミッション・テクノロジーを駆使して開発され、伝送ロスを最小限に抑えるため導体にFurutech α(Alpha)プロセス処理（-196℃での超低温処理＆特殊電磁界処理）を施した導体に銀を調合した究極の銅銀合金線素材を採用。高精度伝送によりクリアな音声を実現！


■三重シールド構造の採用ですぐれた伝送特性と外来ノイズを低減します。
静電気と電磁波防止の二重アルミ箔シールドを施しました。従来のケーブルより遮蔽効果に優れており、ケーブル全体にはα(Alpha)銅導体編組シールドを加え、三重シールドで構成されています。

■厚肉金メッキコネクタ
タイプ端子の金属部はα-処理により導電性が向上し信号伝送ロスが極めて少なく安定性も大幅にアップしました。 接点部は弱磁気性の厚肉金メッキ処理を施し更に音質の向上と長期的な信頼性を確保します。

■制振シース
シースにはRoHS指令適合の振動の影響を軽減させる特殊な柔軟性PVCとナイロンスリーブの効果で内部の共振を防止し、高性能な制振効果を発揮します。
