iShoot 全金属のリング式三脚座、レンズサポート襟、三脚マウントリング for ニコン Nikon PC NIKKOR 19mm F4E ED ティルトシフトレンズ, クイックリリースプレート付き、ARCA-SWISS、PMGタイプの三脚雲台/ボールヘッドクランプ を対応

レーザースキャンテクノロジーを使用してレンズパラメーターを正確に取得し、データに基づいてレンズブラケットの内部形状を正確に刻みます。 一般的に使用されるスイッチとノブの位置を避け、レンズの主な機能に影響を与えない。三脚リングはレンズの前面に取り付けられており、回転できません。 この三脚マウントリングは、他のレンズには使用できません。
レンズサポートは3D彫刻技術を採用しています。 独創的なバンプ設計により、レンズの凹面部分と凸面部分のエッジに固定されます。 分割構造と180度のクイックロックノブを採用しています。 ノブを半回転させて、レンズをしっかりとロックします。正味重量は約160gです。
リング式三脚座の最短内径は約74mmで、レンズに接触するベアリング表面には厚さ0.3mmの3Mプラスチックフィルムが貼られています。 クイックリリースプレートの長さは約50mmで、1/4および3/8のネジ穴とステンレス鋼の滑り止め釘が付いています。
このリング式三脚座は、高密度の航空アルミニウム製です。 世界クラスのハイエンドCNCターニングセンターで正確に機械加工され。中空の彫刻は、ダイヤモンドコンビネーションカッターを使用して実行されました。振動研磨技術の採用により、手動よりも研削精度が20%向上します。 複数の陽極酸化により、金属表面の色の細かさと均一性が10%向上します。
底部のクイックリリースプレートは 39mm アルカスイスタイプで、RRS / ARCA-SWISS / PMG / WIMBERLEY / MARKINS / PHOTOCLAM / SUNWAYFOTO / BENRO / SIRUI / MARSACE / ISHOOT ボールヘッドと互換性があり。
