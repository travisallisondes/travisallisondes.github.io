色：12層 幅広いアプリケーション---新しい12層クラスターケースは、Raspberry Piクラスター、Raspberry Pi NASサーバー、Jetson Nanoクラスター、およびディスクアレイクラスター向けに設計されており、シェル保護および冷却ソリューションを提供します。 Raspberry Pi4B / 3B * / 3B / 2B / B *、2.5MM HDD、Jetson Nanoと互換性があります（注：Raspberry Pi Zeroには対応していません）

簡単な取り付け---長い固定ネジは、スムーズで取り付けが簡単なフラットネジで設計されています。 位置決め穴のフールプルーフ設計により、設置時のトラブルを防ぎます。 仕様の異なるさまざまなネジを使用して、さまざまな製品の固定ニーズに対応できます。 たとえば、Raspberry Pi、2.5インチハードドライブ、Jetson Nanoなどです。

より良い冷却効果---より良い熱放散のための120mm 5V RGB LEDファン付き。このケースは3つの側面が開いており、各アクリル層には、Raspberry Piごとに最大の空気の流れに十分なスペースが含まれています。片側に大きなファンがあり、各Raspberry Piの温度を下げるために強風を流します。

Raspberry Piとの優れた互換性--- GPIOポート、HDMIポート、USBポート、SDカードスロットなど、Raspberry Piボードのすべてのポートにアクセスできます。また、最新のRaspberry Pi 4 Model BおよびRaspberry Pi 3B * / 3B / 2B / B *とも互換性があります。
